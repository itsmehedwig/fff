CSV Format Instructions

Download CSV Template

Download a sample CSV file with the correct format to get started

Your CSV file should have the following columns (in this exact order):

ISBN, Book Name, Author, Date Published, Category, Pieces, Description

Required fields:

  • ISBN - International Standard Book Number (unique identifier)
  • Book Name - Title of the book
  • Author - Author name
  • Category - Book category/genre

Optional fields:

  • Date Published - Year of publication (e.g., 2023)
  • Pieces - Number of copies (defaults to 1 if empty)
  • Description - Book description

Example CSV row:

978-0-123456-78-9,Sample Book Title,John Doe,2023,Fiction,5,This is a sample book description
