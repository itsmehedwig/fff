CSV Format Instructions

Your CSV file should have the following columns:

isbn,title,author,category,publisher,year_published,copies_total,description

Required fields:

  • isbn - International Standard Book Number (unique identifier)
  • title - Book title
  • author - Author name
  • category - Book category/genre

Optional fields:

  • publisher - Publisher name (can be empty)
  • year_published - Year of publication (can be empty)
  • copies_total - Number of copies (defaults to 1 if empty)
  • description - Book description (can be empty)

Example CSV row:

9780134685991,Effective Java,Joshua Bloch,Programming,Addison-Wesley,2018,3,A comprehensive guide to Java programming best practices
